## Marquee Base

A hero section with dual horizontal marquee image galleries on the right side and content on the left, featuring a scramble button effect.

<ComponentWithInstallation 
  component={CtaMarqueeBase}
  scale={0.6}
  componentName="cta-marquee-base"
  componentPath="registry/default/cta-marquee-base.tsx"

  theme: {
    extend: {
      keyframes: {
        "marquee": {
          "0%": { transform: "translateX(0)" },
          "100%": { transform: "translateX(-100%)" }
        }
      },
      animation: {
        "marquee": "marquee var(--duration, 40s) linear infinite"
      }
    }
  }
}`}
  css={`@keyframes marquee {
  0% { transform: translateX(0); }
  100% { transform: translateX(-100%); }
}

.animate-marquee {
  animation: marquee var(--duration, 40s) linear infinite;
}`}
/>

### Usage

