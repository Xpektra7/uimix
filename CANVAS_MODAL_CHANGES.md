# Canvas Component Modal - Новый боковой макет для канваса

## Что изменилось

При клике на компонент в infinite canvas grid теперь открывается новый модальный компонент `CanvasComponentModal` с боковым макетом:

- **Слева**: Код компонента и секция установки через регистр
- **Справа**: Превью компонента
- **Сверху от кода**: CLI команда и инструкции по установке

## Технические изменения

### 1. Создан новый компонент `CanvasComponentModal`
- Заменил `ScaledIframePreview` в модальном окне канваса
- Использует `ComponentPreview` и `ComponentCode` для отображения
- Автоматически определяет путь к компоненту на основе его имени
- Загружает код компонента через API `/api/code`

### 2. Обновлен `infinite-canvas-grid.tsx`
- Удален компонент `ScaledIframePreview` (больше не используется)
- Удален импорт `react-frame-component`
- Удалена переменная `modalScale`
- Добавлен импорт `CanvasComponentModal`
- Заменено модальное окно на новый компонент

### 3. Маппинг компонентов
Создан автоматический маппинг имен компонентов на пути к файлам:
```typescript
const nameMap: Record<string, string> = {
  'Hero Dock': 'registry/default/hero-dock.tsx',
  'Hero Minimalism': 'registry/default/hero-minimalism.tsx',
  // ... и так далее
};
```

## Использование

Теперь при клике на любой компонент в канвасе:
1. Открывается модальное окно с боковым макетом
2. Слева отображается код компонента и инструкции по установке
3. Справа отображается превью компонента
4. Можно закрыть модальное окно кнопкой ✕

## Совместимость

- Все существующие компоненты в канвасе автоматически поддерживают новый макет
- Не требует изменений в registry или других файлах
- Обратно совместим с существующим API
